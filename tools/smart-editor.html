Good to know

  • Your work is auto-saved in your browser every 2 seconds. Closing and reopening the page will restore your last session.
  • Use A− / A+ to adjust the text size for your comfort — applies to the whole document and is remembered next time you open the editor. Use − / + next to the number to control line spacing (1.2 = tight, 1.8 = default, 2.8 = very airy). Both settings are saved automatically.
  • Writer mode is for notes, drafts, articles, and any text where formatting matters. Use the toolbar to apply headings, bold, lists, and more. Enable Focus mode to dim everything except your current paragraph.
  • Code mode gives you syntax highlighting for HTML, JavaScript, CSS, Python, and SQL, along with line numbers, bracket matching, and auto-closing brackets. Perfect for editing HTML files or writing quick code snippets.
  • Switching between modes transfers your content as plain text. If you switch from Writer to Code, formatting (bold, headings etc.) will be lost — you'll be warned before this happens.
  • Use Open file to load any .txt, .html, .js, .css, .py, or other text file directly from your computer.
  • PDF export is available in Writer mode. DOCX export requires ✦ Premium.
  • Typewriter mode (✦ Premium) keeps your current line vertically centered as you type — reduces eye movement and helps you stay in flow.
