Good to know

  • Everything happens in your browser — your files are never uploaded anywhere.
  • Supported formats: plain text (.txt), Markdown (.md), Word documents (.docx), and PDFs (.pdf).
  • Images inside .docx or .pdf files are skipped — only the text is extracted and merged.
  • Drag the file cards to reorder them before exporting. The output follows the order shown.
  • Right-to-left text (Hebrew, Arabic) is detected automatically and preserved in Word and plain text exports.
  • PDF and special characters: By default, PDF uses a basic font that cannot render Hebrew, Arabic, or symbols like arrows. If your files contain these, enable the Unicode font option that appears in settings when PDF is selected — it loads a ~400KB font once per session and renders everything correctly. Alternatively, export to Word (.docx) which supports all characters natively with no extra download.
  • Completely blank files are skipped and you will see a warning for each one. Files with very little content are included as-is.
  • Password-protected Word or PDF files cannot be read. Remove the password first, then re-upload.
  • Each file can optionally start on a new page — use the toggle on each file card.
