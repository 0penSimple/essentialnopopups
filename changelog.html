Update April 2026
Faster, smarter, and finally works on your phone

A big under-the-hood update focused on speed and reliability. Video and audio tools are dramatically faster thanks to hardware-accelerated processing, and nearly everything now works properly on iOS and Android.

  • New Color Palette & Picker — extract a full color palette from any image, or pick exact pixel colors with an eyedropper. Perfect for designers and developers.
  • Improved Video converter is now dramatically faster — MP4, MOV, WebM, and MKV conversions now use hardware-accelerated encoding via the browser's native WebCodecs API instead of software emulation. A 30-second clip that used to take 10+ minutes now takes seconds.
  • Improved iPhone and iPad support across all tools — every FFmpeg-powered tool (video converter, trim video, remove audio, extract audio, audio converter, trim audio) now works on iOS Safari. Previously they'd silently fail or hang.
  • Improved Downloads on iPhone — tapping download now brings up the native iOS share sheet so you can save directly to Files, Photos, or share via AirDrop. No more blank white pages.
  • Improved Re-download without re-processing — after a tool finishes, the output button stays active so you can download the result again at any time without re-running the operation.
  • Fixed Video to GIF and Extract Audio — both tools were silently failing on certain files due to a memory conflict. Fully resolved.
  • Improved Processing feedback — instead of a frozen progress bar stuck at 0%, tools now show a live animated bar and rotating status messages while working.
Launch March 2026
EssentialBits is live

After months of building, EssentialBits launches with 25+ free browser-based tools across video, images, audio, PDF, and productivity. No sign-up. No pop-ups. Nothing uploaded.

  • New Smart Editor — a writer's editor and full code editor in one. Writer mode has formatting, focus mode, and typewriter mode. Code mode has syntax highlighting for HTML, JS, CSS, Python, and SQL. Auto-saves to your browser, no cloud, no account.
  • New QR Code Generator — generate QR codes with logo overlay, custom colors, and shaped frames (circle, octagon, shield, folded corner). Premium users unlock custom module shapes and batch generation.
  • New Notes Merger — merge multiple .txt, .md, Word, and PDF files into one document. Export as plain text, PDF, or Word.
  • New Find & Replace — search and replace across large text blocks with case-sensitive matching and live preview.
  • New 25+ tools across Video, Images, Audio, and PDF — compress, convert, trim, merge, split and more. All free, all in your browser.
